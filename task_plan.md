## 关键问题
1. 超市 API 接口地址可能已变更，需要使用浏览器开发者工具验证
2. 爬虫可能被超市网站限流或封禁
3. Countdown API 可能需要特殊的请求头或认证

## 已做决策
| 决策 | 理由 |
|------|------|
| 使用 Next.js (App Router) | 支持 API Routes + 前端一体，部署简单 |
| 使用 TypeScript | 类型安全，减少运行时错误 |
| 使用 Tailwind CSS | 快速构建响应式 UI |
| 先做实时爬取，后续再加数据库缓存 | 快速验证 MVP |
| 采用 PWA 方案而非原生 App | 开发成本低，跨平台 |
